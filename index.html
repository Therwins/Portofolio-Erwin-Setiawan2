S1 Teknologi Informasi

Universitas Amikom Yogyakarta | 2024 – Sekarang

Saya merupakan mahasiswa Program Studi Teknologi Informasi dengan fokus pembelajaran pada pemrograman web dan animasi 3D. Selama masa perkuliahan, saya mempelajari konsep serta penerapan teknologi informasi yang berkaitan dengan pengembangan aplikasi berbasis web, perancangan antarmuka pengguna, serta pembuatan visual digital dan animasi tiga dimensi.

Pada bidang pemrograman web, saya mempelajari dan menerapkan pembuatan website mulai dari struktur dasar, desain antarmuka, hingga pengelolaan data menggunakan teknologi web modern. Saya terbiasa mengerjakan proyek yang mencakup perancangan UI, implementasi front end, serta integrasi dengan sistem basis data.

Sementara itu, pada bidang animasi dan visualisasi 3D, saya mempelajari proses pembuatan objek dan environment 3D, meliputi modeling, texturing, pencahayaan, dan rendering. Proyek-proyek yang dikerjakan membantu saya memahami bagaimana visual 3D dapat digunakan untuk media interaktif, presentasi, maupun kebutuhan kreatif lainnya.

Melalui kombinasi antara kemampuan teknis web development dan kreativitas di bidang 3D, saya terbiasa menghasilkan karya digital yang tidak hanya fungsional, tetapi juga memiliki nilai estetika. Saya terus mengembangkan kemampuan melalui proyek akademik, latihan mandiri, serta pembuatan portofolio sebagai persiapan untuk berkarier di bidang Front End Development, Creative Developer, atau 3D Artist.
